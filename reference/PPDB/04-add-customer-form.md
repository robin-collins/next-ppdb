# Add Customer Form Documentation

## Page URL

`http://10.10.10.44/ppdb/add_customer.php`
![[after-add-customer-click.png]]

## Page Title

"Add Customer"

## Access Method

Accessed by clicking the "Add Customer" button from the main search interface.

## Form Structure

The Add Customer form contains the following fields for entering new customer information:

### Customer Information Fields

#### Personal Details

- **Firstname**: Text input field for customer's first name
- **Surname**: Text input field for customer's surname/last name

#### Contact Information

- **Address**: Text input field for street address
- **Suburb**: Text input field for suburb/city
- **Postcode**: Text input field for postal code

#### Phone Numbers

- **Phone1**: Primary phone number text input
- **Phone2**: Secondary phone number text input
- **Phone3**: Tertiary phone number text input

#### Digital Contact

- **Email**: Email address text input field

### Form Actions

- **Insert Record**: Button to submit and save the new customer record

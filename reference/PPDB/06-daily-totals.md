# Daily Totals Documentation

## Access Method

Accessed by clicking the "Daily Totals" button from the main search interface within the frameset.

## Functionality Overview

The Daily Totals feature provides a simple end-of-day summary report displaying key business metrics for the current date.

## Business Purpose

This feature serves several operational needs:

1. **Daily Reconciliation**: Quick verification of day's activity
2. **Revenue Tracking**: Immediate visibility into daily earnings
3. **Activity Monitoring**: Count of animals serviced
4. **End-of-Day Reporting**: Simple summary for business records

## Technical Notes

- **Real-time Data**: Shows current day's accumulated totals
- **Simple Interface**: No complex navigation or drill-down options
- **Frameset Integration**: Displays within the main application structure
- **Timestamp Accuracy**: Precise time recording for report generation

## Operational Context

- **Zero Values**: Screenshot taken during non-business hours or slow day
- **Live Data**: Would typically show accumulated values throughout business day
- **Staff Usage**: Likely checked by staff at end of shifts or day

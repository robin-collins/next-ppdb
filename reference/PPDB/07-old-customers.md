# Old Customers Documentation

## Access Method

Accessed by clicking the "Old Customers" button from the main search interface within the frameset.

## Functionality Overview

The Old Customers feature displays a comprehensive historical database of past customers, providing staff with access to previous client information and visit history.
![[old-customers-result.png]]

## Database Table Structure

### Column Layout

The customer database displays the following columns:

1. **Name**: Customer's full name (surname, first name format)
2. **Address**: Complete street address including suburb/city and postal code
3. **Phone**: Customer contact phone number
4. **Animal**: Pet name and breed (format: "Pet Name - Breed")
5. **Last Visit**: Date of most recent visit (YYYY-MM-DD format)

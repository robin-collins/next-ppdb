# Next PPDB Documentation

Welcome to the **Pampered Pooch Database** (Next PPDB) documentation. This modern pet grooming management system helps you track customers, animals, breeds, and grooming services.

![Home Search Screen](/docs/screenshots/home-empty.png)

## What is Next PPDB?

Next PPDB is a comprehensive pet grooming database application built with Next.js 15. It features:

- **Smart Search** - Find animals by name, breed, owner, phone number, or email with relevance-based ranking
- **Customer Management** - Track customer information, contact details, and associated pets
- **Animal Records** - Maintain detailed grooming history and service notes for each animal
- **Breed Database** - Manage breeds with customizable pricing and time estimates
- **Reports** - Generate daily totals and track customer activity
- **Database Backup** - Create and download database backups

## Key Features

### Glassmorphic UI Design

The application features a modern glassmorphic design with animated gradients, providing a clean and professional appearance for daily use.

### Relevance-Based Search

The search system uses intelligent scoring to rank results by relevance:
- Exact matches score highest
- Partial matches and fuzzy matching help find results even with typos
- Phone numbers are normalized for flexible searching

### Service History Tracking

Every grooming service is recorded with timestamps and technician initials, building a comprehensive history for each animal.
